这是什么？

mediatools 是一套自用媒体自动化工具集，目前包含：

  • emby-inspector：Emby 近期播放/库增长/任务中心（转存/整理/STRM/刷新等流水线）的控制台。
  • telegram-talk-bot：Telegram 双向沟通/转发（可对接入队）。
使用提示
  • 「设置」里配置 Emby Base URL 与 API Key 后，首页/最近播放/库增长会自动拉取数据。
  • 任务中心用于观察 TG 入队与流水线执行日志。
  • 账号/密码修改也在「设置」页面内。
